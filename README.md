Blog Management REST API
Overview

A production-ready RESTful Blog Management API built with Node.js, Express.js, and MongoDB following REST architecture.

The project supports authentication, blog management, categories, image uploads, search, pagination, and Docker-based deployment.

This project was developed as part of my MERN learning journey and further enhanced with additional improvements, project restructuring, and Docker containerization.
